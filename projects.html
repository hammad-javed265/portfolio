Water Purity System | IIOT | Fiverr Projects


I develop water purity monitoring systems for my clients. I gather data from sensors via gateways and transmit it to the cloud. Through the cloud, I retrieve the data using an MQTT broker, and then I create software (a dashboard) to display this data in charts, tables, and alarms. And I also make the User Management System In it.


Customized Dashboard


A fully customized dashboard that allows users to create dynamic pie charts and bar charts effortlessly. This powerful tool combines JavaScript, PHP, HTML, CSS, and leverages the capabilities of MySQL and MongoDB databases.
• The core functionality of my dashboard lies in its ability to adapt to user input. By simply entering meters, parameters, color and time, the dashboard dynamically adjusts the height and width of the chart elements, ensuring a personalized visual experience.
• One of the key features I'm particularly proud of is the chart persistence. Even after refreshing the page, the charts retain their state, allowing users to revisit their data effortlessly. This ensures continuity and allows for easy analysis and comparison over time.
• Another highlight of my customized dashboard is its flexibility. Users can create an unlimited number of charts, tailoring each one to their specific needs. The intuitive interface ensures a seamless user experience, making data visualization a breeze.
